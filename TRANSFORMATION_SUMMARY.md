## 🎓 Usage Examples

### Quick Scan
```bash
python main.py scan --csv data.csv --table customers
```

### With Custom Checks
```bash

```

### Generate Report
```bash

```

### Start API Server
```bash
python main.py api --port 8000
```

### View History
```bash
python main.py history --table customers --days 30
```

### View Trends
```bash
python main.py trends --table customers --days 7
```
